# Компания «Карусель» — премиальный лендинг

Одностраничный корпоративный сайт для образовательной компании: строгий, технологичный стиль, 3D-карусель в hero, затемняющий градиент, KPI, галерея с лайтбоксом, видео и блок директора.

## Структура

- `index.html` — разметка всех секций
- `styles.css` — тёмная палитра (чёрный, графит, серый, белые акценты), адаптив, анимации
- `script.js` — 3D-карусель (лошади/кресла), параллакс по мыши, лайтбокс, анимации при скролле, мобильное меню

## Секции

1. **Hero** — 3D-карусель (desktop) или видео с каруселью (mobile), градиент, «Стать партнёром»
2. **О компании** — краткий деловой текст
3. **Достижения** — KPI: 10+ лет, 1000+ учеников, 40+ филиалов, спонсор аренды, до 1800 зрителей
4. **Нам доверяют** — детские сады, школы, родители
5. **Направления** — хореография, футбол, математика, подготовка к школе, вокал (фото + hover)
6. **Галерея** — мероприятия и фестивали, лайтбокс по клику
7. **Видео** — широкий блок, autoplay muted
8. **Директор** — фото, имя, должность, цитата
9. **CTA** — «Связаться с нами», телефон, email

## Запуск

```bash
cd /Users/user/Desktop/karusel
python3 -m http.server 8787
```

Открыть: `http://localhost:8787`

## Замена контента

- **Видео hero и блока «О нас»**: заменить `src` в тегах `<video>` (сейчас — Mixkit).
- **Фото направлений и галереи**: заменить `src` в `img` (сейчас — Unsplash).
- **Директор**: фото, имя, должность и цитату в `#director`.

## Технологии

- HTML5, CSS3 (Grid, Flexbox, 3D transforms)
- Vanilla JS: `requestAnimationFrame` для карусели, `IntersectionObserver` для появления блоков

## Адаптив

- **Desktop (≥1024px)**: 3D-карусель в hero, параллакс.
- **Tablet / Mobile**: в hero — только видео, 3D отключён; меню-бургер, упрощённые сетки.
