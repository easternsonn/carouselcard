# Компания «Карусель» — премиальный лендинг

Одностраничный корпоративный сайт для образовательной компании: строгий деловой стиль, полноэкранный логотип в hero, бегущая лента новостей, KPI, галерея с лайтбоксом, видео и блок директора.

## Структура

- `index.html` — разметка всех секций
- `styles.css` — тёмная палитра (чёрный, графит, серый, белые акценты), адаптив, анимации
- `script.js` — лайтбокс, анимации при скролле, мобильное меню, отправка форм на email
- `server.js` — сервер отправки заявок на email (Node.js + Nodemailer)
- `EMAIL_SETUP.md` — инструкция по настройке email

## Секции

1. **Hero** — полноэкранный логотип, градиент, «Стать партнёром»
2. **Новости компании** — крупные карточки с фото, бегущая лента
3. **О компании** — краткий деловой текст
4. **Достижения** — KPI: 10+ лет, 1000+ учеников, 40+ филиалов, спонсор аренды, до 1800 зрителей
5. **Нам доверяют** — детские сады, школы, родители
6. **Направления** — хореография, футбол, математика, подготовка к школе, вокал (фото + hover)
7. **Галерея** — мероприятия и фестивали, лайтбокс по клику
8. **Видео** — широкий блок, autoplay muted
9. **Директор** — фото, имя, должность, цитата
10. **CTA** — «Связаться с нами», телефон, email

## Запуск

```bash
python3 -m http.server 8787
```

Открыть: `http://localhost:8787`

## Замена контента

- **Логотип hero**: заменить файл `logo/carousel.png`.
- **Фото новостей**: заменить `src` в блоке `#news`.
- **Фото направлений и галереи**: заменить `src` в `img`.
- **Директор**: фото, имя, должность и цитату в `#director`.

## Технологии

- HTML5, CSS3 (Grid, Flexbox)
- Vanilla JS: `IntersectionObserver` для появления блоков

## Адаптив

- **Desktop (≥1024px)**: полноэкранный логотип и лента новостей.
- **Tablet / Mobile**: меню-бургер, упрощённые сетки.
